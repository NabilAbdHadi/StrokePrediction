1) id: unique identifier
2) gender: {
    0: "Male", 
    1: "Female", 
    2: "Other"
    }
3) age: age of the patient
4) hypertension: 
    0 if the patient doesn't have hypertension, 
    1 if the patient has hypertension
5) heart_disease: 
    0 if the patient doesn't have any heart diseases, 
    1 if the patient has a heart disease
6) ever_married: 
    0 if "No",
    1 if "Yes"
7) work_type: {
    0 : "Never_worked", 
    1 : "children", 
    2 : "Govt_job", 
    3 : "Private", 
    4 : "Self-employed" 
    }
8) Residence_type: {
    0: "Rural", 
    1: "Urban"
    }
9) avg_glucose_level: average glucose level in blood
10) bmi: body mass index
11) smoking_status: { 
    0 : "never smoked", 
    1 : "formerly smoked", 
    2 : "smokes", 
    3 : "Unknown"
    }
12) stroke: 
    1 if the patient had a stroke 
    0 if does not have stroke
*Note: "Unknown" in smoking_status means that the information is unavailable for this patient
